# Uso
1. Recuerda revisar a detalle el video para no tener problemas a la hora de instalar Manim.
2. Modifica el archivo Algebra.txt cuidando que el codigo LaTex esté entre cuatro "$", de lo contrario el script de algebra no mostrará nada.
3. Tambien dejé el código que usé para la miniatura xd.
4. Para cualquiera de los códigos que elijas el renderizado comienza cuando le das al botón del plugin instalado en el VS Code.
